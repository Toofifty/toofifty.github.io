Usage

Options marked with a * may only work proeperly after a refresh

Point grid* - toggle between random particle positions and an aligned grid.

Top-down view* - toggle between top-down view or side view.

Colours* - toggle between all white or randomly coloured particles.

Wave rate - change the speed/direction of the resonating effect.

Height mult - change the average height of the particles (height is out of the screen).

Max height - cap the maximum height that can be reached by the particles.

Distance coefficient - alter the (inverse) relationship between the height of the particle and it's distance from the centre point.

Spread coefficient - alter the amount that spread affects height.

X white space* - change the width of the animation.

Y white space* - change the height of the animation.

Anti-spike - increase to remove stray particles that travel too high. This also reduces height of all waves.

# X points* - change the number of X (horizontal) points in the point grid. For random points, # points = #X * #Y.

# Y points* - change the number of Y (vertical) points in the point grid.

Alpha fade - change the amount the particles fade when in fade type 0 (height-based). Setting to 255 will remove all fading.

Height exponent - exponentially change how many times the distance is used to reduce the wave height.

Fade type* - toggle between the 3 states of fade - height based (0), radius based (1), and random (2) (WIP).
